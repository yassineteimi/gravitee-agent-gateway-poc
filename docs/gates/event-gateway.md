# Gate 3: Event Gateway

The event-native differentiator: govern a native Kafka stream as a first-class API, with a Plan, authentication mediation, and protocol mediation to HTTP, SSE or WebSocket.

!!! info "In progress"
    Written during **Milestone 3**. Exposes a Kafka topic through the Gravitee Kafka Gateway, secures it with a Plan and authentication mediation, applies access control, and shows protocol mediation so an HTTP or WebSocket client consumes the stream with no native Kafka client. This gate has no Traefik equivalent.
